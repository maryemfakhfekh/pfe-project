# PFE Project - Gestion des Stagiaires

## Description
Application de gestion des stagiaires composée d'un backend Spring Boot,
d'une application mobile Flutter et d'une interface web Angular.

## Repos
- 🟠 [Backend Spring Boot](https://github.com/maryemfakhfekh/gestion-stagiaires-backend)
- 🟢 [Mobile Flutter](https://github.com/maryemfakhfekh/gestion_stagiaire_mobile)
- 🔵 [Frontend Angular](https://github.com/maryemfakhfekh/gestion-stagiaires-web)

## Technologies
- **Backend** : Spring Boot, Java
- **Mobile** : Flutter, Dart
- **Frontend** : Angular, TypeScript
